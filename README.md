# Dare Mighty Things 2019: Machine Learning Solution for JLL

## Overview

Dare Mighty Things 2019 is a hackathon project focused on applying machine learning to solve a real-world business problem for JLL (Jones Lang LaSalle).

The system processes structured and unstructured data from spreadsheets and text inputs to generate insights using classical machine learning and natural language processing techniques.

This project was developed prior to modern LLM-based AI systems and reflects an early exploration of applied machine learning workflows.

---

## Problem

Enterprises often deal with:

* Large volumes of spreadsheet-based data
* Unstructured text inputs that are difficult to analyze
* Manual data processing workflows that are time-consuming

These challenges limit the ability to extract useful insights efficiently.

---

## Solution

This project:

* Processes spreadsheet data using Python and pandas
* Applies natural language processing techniques to text inputs
* Uses machine learning models to analyze and generate structured outputs
* Serves results through a backend system for further use

The system demonstrates how traditional ML pipelines can be used to automate data processing and analysis.

---

## Key Features

* Data ingestion from spreadsheet sources
* Data transformation using pandas DataFrames
* Natural language processing using Java-based components
* Machine learning model integration (serialized `.pkl` models)
* Backend system for processing and serving results

---

## System Architecture

1. Input → spreadsheet and text data
2. Data processing → pandas DataFrame transformations
3. NLP layer → Java-based text processing
4. ML models → trained and stored as `.pkl` files
5. Backend server → Python-based processing and execution
6. Output → structured insights
